# 2026-Masters-Thesis-Human-Bacteroidota-Species-on-Dietary-Fibre-in-Pure-Mixed-Culture
Statistical analysis for masters thesis entitled:  Growth Characteristics of Human Intestinal Bacteroidota Species on Dietary Fibre Polysaccharides in Pure &; Mixed Culture. Analysis was conducted in R using linear modeling, anova, TukeyHSD, and estimated linear means was conducted.
All work was conducted by Ellie Campbell at the Rowett Insititue, University of Aberbeen.
